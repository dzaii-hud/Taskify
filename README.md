# Taskify

Taskify adalah aplikasi manajemen proyek dan tugas berbasis web yang memudahkan tim dalam mengelola proyek, memantau progress tugas, dan berkomunikasi secara efisien. Sistem ini menggunakan Laravel, Livewire, dan Tailwind CSS untuk memberikan pengalaman interaktif dan responsif.

---

## Fitur Utama
- **Dashboard User & Admin** untuk monitoring proyek dan tugas
- **Manajemen Proyek:** CRUD proyek, status otomatis
- **Manajemen Tugas:** CRUD tugas, penugasan ke user, status On Going / Done
- **Statistik:** Progress proyek dan tugas per user dan seluruh tim
- **Notifikasi:** Realtime saat tugas dibuat atau status berubah
- **Role & Permissions:** Admin dan User dengan akses berbeda
- **Keterbatasan:** Fitur pembentukan tim, kalender, dan ekspor PDF belum dikembangkan karena keterbatasan waktu, hak akses admin juga belum sepenuhnya menyeluruh

---

## Teknologi yang Digunakan
- Laravel 12
- Livewire
- Tailwind CSS
- Chart.js (untuk grafik statistik)
- MySQL (database)
- Git & GitHub (version control)
